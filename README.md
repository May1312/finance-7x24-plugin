# 东方财富 7x24 快讯 IntelliJ 插件

在 IntelliJ IDEA 中查看东方财富 7x24 小时财经快讯。

## 功能

- 在 IDEA 右侧工具窗口展示财经快讯列表
- 启动后自动加载最新快讯
- 每 30 秒自动检查并刷新新消息
- 支持手动刷新
- 快讯内容支持自动换行

## 安装

### 从 JetBrains Marketplace 安装

插件发布后，可在 IDEA 的 `Settings` -> `Plugins` 中搜索安装。

### 从本地文件安装

运行以下命令构建插件包：

```bash
./gradlew buildPlugin
```

然后在 IDEA 中选择 `Settings` -> `Plugins` -> 齿轮图标 -> `Install Plugin from Disk...`，选择 `build/distributions/` 下生成的 zip 文件。

## 兼容性

- IntelliJ IDEA 2024.1 及以上版本
- 基于 IntelliJ Platform 构建

## 数据来源

插件展示的数据来自东方财富公开财经快讯接口。数据内容、更新频率和可用性取决于数据来源服务。
