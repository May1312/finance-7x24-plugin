# Finance 7x24 News IntelliJ Plugin

Finance 7x24 News 是一个 IntelliJ IDEA 工具窗口插件，用于在 IDE 内查看 7x24 财经快讯，并维护股票、ETF、指数自选行情。

## 功能

- 在 IDEA 右侧工具窗口展示 7x24 财经快讯
- 快讯支持自动刷新、手动刷新和内容自动换行
- 提供自选行情标签页，支持股票、ETF 和指数代码
- 自选行情支持添加、右键删除、手动刷新和 30 秒自动刷新
- 支持按表头排序，连续点击可在倒序、正序、默认顺序间切换
- 行情接口支持多数据源兜底，优先使用东方财富，失败时尝试腾讯财经和新浪财经
- 行情涨跌使用 A 股习惯配色：上涨红色，下跌绿色

## 安装

### 从 JetBrains Marketplace 安装

插件发布后，可在 IDEA 的 `Settings` -> `Plugins` 中搜索安装。

### 从本地文件安装

运行以下命令构建插件包：

```bash
./gradlew buildPlugin
```

然后在 IDEA 中选择 `Settings` -> `Plugins` -> 齿轮图标 -> `Install Plugin from Disk...`，选择 `build/distributions/` 下生成的 zip 文件。

## 兼容性

- IntelliJ IDEA 2024.1 及以上版本
- 基于 IntelliJ Platform 构建

## 数据来源

插件展示的财经快讯来自东方财富公开财经快讯接口。自选行情优先使用东方财富公开行情接口，并在请求失败时尝试腾讯财经和新浪财经公开行情接口。

公开接口的数据内容、更新频率、实时性和可用性均取决于对应数据来源服务，插件不会保证行情数据实时、完整或持续可用。
