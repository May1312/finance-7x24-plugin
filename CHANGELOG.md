# Changelog

## 1.0.1

- 自选行情红绿配色统一，修复同一行半红半绿
- 快讯页换行预留滚动条宽度，避免侧边栏收窄后文字溢出
- 自选列表优化：去除涨跌额列、列不隐藏、名称列优先显示
- 工具栏与工具窗口图标加品牌色
- 闭市时暂停行情轮询（保留最近行情，首次打开仍加载）

## 1.0.0

- 初始版本发布
- 支持 7x24 财经快讯列表展示
- 支持自动刷新和手动刷新
- 支持快讯内容自动换行
- 新增自选行情标签页，支持股票、ETF 和指数代码
- 支持自选代码添加、右键删除、手动刷新和自动刷新
- 支持表格按列三态排序：倒序、正序、默认顺序
- 行情数据支持多数据源兜底（含东方财富、腾讯财经、新浪财经等）
- 新增插件图标和工具窗口图标
- 支持 IntelliJ IDEA 2024.1 及以上版本
